### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/luckybulgur/fivem-dashboard.git
   ```
2. Install packages
   ```sh
   yarn
   ```
3. Create a `.env` in the backend
   ```
MYSQL_PASSWORD=""
MYSQL_HOST=""
MYSQL_USER=""
MYSQL_PASSWORD=""
MYSQL_DATABASE=""
# JWT
SECRET_KEY=""
   ```
